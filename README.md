# Installation:

Maven:

```
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>

	...

    	<dependency>
    	    <groupId>com.github.vladrus13</groupId>
    	    <artifactId>JGraphic</artifactId>
    	    <version>Tag</version>
    	</dependency>
	```
